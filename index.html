Comment installer ?

Suivez ces étapes simples pour démarrer.

  1. Cliquez sur le bouton « Télécharger » ci-dessus. Vous serez redirigé vers la page des releases GitHub.
  2. Téléchargez le fichier correspondant à votre système : .exe pour Windows, .dmg pour macOS, ou .AppImage pour Linux.
  3. Une fois le fichier téléchargé, ouvrez-le et suivez les instructions d'installation.
  4. Lancez l'application et profitez-en !
