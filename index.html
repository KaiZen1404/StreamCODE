Comment installer ?

Des instructions simples pour chaque plateforme.

Android : ouvrez l'APK et autorisez l'installation Windows : exécutez l'installateur .exe macOS : ouvrez le .dmg et glissez dans Applications Linux : rendez le .AppImage exécutable TV : installez via le store ou sideload
Vous ne trouvez pas votre version ? Rendez-vous sur la page Releases GitHub pour accéder à tous les fichiers disponibles. Si votre plateforme n'est pas encore prise en charge, vous pouvez ouvrir une issue pour suivre l'avancement.
